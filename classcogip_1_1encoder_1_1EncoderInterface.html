    Note
    Encoder pulse per revolution must be set according to chosen mode eg. For a 2500 PPR encoder:
    • ENCODER_MODE_X1 -> pulse_per_rev = 2500
    • ENCODER_MODE_X2 -> pulse_per_rev = 5000
    • ENCODER_MODE_X4 -> pulse_per_rev = 10000
