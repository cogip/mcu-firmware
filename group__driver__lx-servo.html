    LX servomotors driver

    LX servomotors device driver. More...

    Detailed Description

    LX servomotors device driver.

    LX servomotors uses one-wire serial half-duplex protocol. LX-15D and LX-16A are well known servomotors working with this driver, however this API should work with all LX HiWonder servomotors.

    This driver is inspired by Feetech driver written by Lo√Øc Dauphin loic..nosp@m.daup.nosp@m.hin@i.nosp@m.nria.nosp@m..fr
