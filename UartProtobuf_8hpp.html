    Detailed Description

    Exchange Protobuf messages over UART module.

    This module provides a class that controls an UART to send and receive Protobuf messages using EmbbededProto. A callback function must also be provided to handle decoded messages. A file uartpb_config.hpp must be provided by the application, either at lib, platform or application level. This file must define the PB_OutputMessage type. This type refers to a Protobuf message that contains a oneof attribute grouping all message types that the application and its libraries can send on serial port. See examples/uartb.
